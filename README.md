# 🧱 Tiles Gallery

A modern, aesthetic tile showcase web application built with Next.js.  
Users can explore premium tile designs, view details, and manage their profiles with a smooth, glassmorphism-inspired UI.

---

## 🌐 Live Website
👉 https://tiles-gallery-lingkon.vercel.app/

---

## 🎯 Purpose
The purpose of this project is to demonstrate a full-stack web application with authentication, protected routes, and a visually polished UI using modern web technologies.

---

## ✨ Key Features

- 🔐 User Authentication (Email/Password & Google Login)
- 🧑‍💼 User Profile & Update Information Feature
- 🔍 Search tiles by title
- 🧱 View all tiles and detailed tile pages
- 🔒 Protected routes (profile, update, tile details)
- 🎨 Glassmorphism UI with modern design
- ⚡ Smooth animations using React Spring
- 📱 Fully responsive design

---

## 🛠️ Technologies & Packages Used

### Core
- Next.js
- React
- Tailwind CSS

### Authentication & Database
- Better Auth
- MongoDB

### UI & UX
- Hero UI
- React Spring (animations)
- React Toastify (notifications)
